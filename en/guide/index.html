环境支持 #

由于 Vue 3 不再支持 IE11，Element Plus 也不再支持 IE11 浏览器。

版本 #

LongQiUI 目前还在开发中

使用包管理器 #

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 LongQiUI, 然后您就可以使用打包工具，例如 Vite 和 webpack

# 选择一个你喜欢的包管理器


# NPM
$ npm install longqi-ui --save


# Yarn
$ yarn add longqi-ui


# pnpm
$ pnpm add longqi-ui
